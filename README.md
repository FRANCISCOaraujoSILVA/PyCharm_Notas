- Essas notas foram criadas durante o meu curso de Python 'DO BÁSICO AO AVANÇADO' e nelas estão contidas observações

sem indentação, para corrigir faça:
    - CTRL + ALT + SHIFT + L, e selecione a opção: Whole file
